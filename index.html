Pointer for Block Element

  1. Width of viewport is 100%of viewport
  2. Height of block isnt set by default
  3. Height and width can be explicitly set using CSS

Ponters for Inline Element

  1. Width of inline level element is dependant on the content
  2. Height of inline isn't set by default
  3. Height and width can not be set explicitly by using CSS
  4. Inline element does not work as wrapper for block elements
